# Kenney CC0 Audio Subset

This folder contains a curated subset of Kenney audio packs used by TRENCHFALL.
Only the files referenced by `src/audio/sfxManifest.js` are checked in.

Sources:

- Impact Sounds: https://www.kenney.nl/assets/impact-sounds
- Interface Sounds: https://kenney.nl/assets/interface-sounds
- RPG Audio: https://kenney.nl/assets/rpg-audio

License:

- Creative Commons CC0 1.0 Universal
- https://creativecommons.org/publicdomain/zero/1.0/

Notes:

- Gunshots remain synthesized in-game by design.
- This subset is used for footsteps, physical impacts, UI, reload handling, and knife sounds.
